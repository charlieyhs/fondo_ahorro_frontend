
===== Consejos generales adicionales ===============

    * Manejo de autenticación:
        - Considera usar un contexto de autenticación
        - Implementa un sistema de tokens JWT con refresh tokens
        - Agrega manejo de errores en el login

    * Optimización de rendimiento:
        - Usa React.memo para componentes puros
        - Considera lazy loading para rutas
        - Usa useCallback para funciones pasadas como props

    * Accesibilidad:
        - Agrega roles ARIA
        - Implementa manejo de foco
        - Usa etiquetas semánticas correctamente

    * Seguridad:
        - Nunca almacenes tokens directamente en localStorage
        - Considera usar HttpOnly cookies
        - Implementa protección contra CSRF

    * Pruebas:
        - Agrega pruebas unitarias con Jest
        - Implementa pruebas de integración
        - Considera pruebas E2E con Cypress

======== Flujo aplicación back-end ==================
sequenceDiagram
    participant Frontend (React)
    participant Backend (Spring)
    Frontend->>Backend: POST /login (credenciales)
    Backend-->>Frontend: JWT Token
    Frontend->>AuthProvider: Almacena token y estado
    AuthProvider->>Frontend: Actualiza UI
    Frontend->>Backend: Solicitudes con token en headers
    Backend-->>Frontend: Respuesta autenticada
